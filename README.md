# InvestIQ – AI-Enhanced Personal Finance Advisor

## Overview
A Python command-line app that analyzes your financial profile, suggests investments based on your risk tolerance, and uses machine learning to forecast when you'll hit your savings goal.

## Features
- Budget analysis with savings rate calculation
- Personalized investment suggestions (conservative / moderate / aggressive)
- ML-powered savings goal forecasting using scikit-learn
- Saves user profile locally as JSON

## How to Run
1. Clone the repo
2. Install dependencies: pip install -r requirements.txt
3. Run: python Main.py

## Tools Used
- Python
- pandas
- scikit-learn
- NumPy

## Data
- stock_data.csv — investment options with risk levels and average returns
- user_data.json — saved user profile
